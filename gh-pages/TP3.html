Recherche des états stationnaires (ES)

Si l’on a une équation à une seule variable

\[ \frac{dx}{dt} = f(x, p_k) \]

où \(p_k\) est l’ensemble des paramètres de l’ED, les états stationnaires, s’ils existent seront identifiés par les valeurs \(x^\star\) ? annulant la fonction \(f : f (x^\star, p_k ) = 0\)

Soit une équation différentielle

\[ \frac{dx}{dt} = f_1(x) - f_2(x) \] où

\[ f_1(x) = k_0\left(1 + \frac{1}{x}\right) \]

\[ f_2(x) = k_1x^\alpha \]

avec \(k_0 = 10\), \(k_1 = 150\) et \(\alpha = 1.5\)

k0 <- 10; k1 <- 150; a <-1.5
f1 <- function(x) k0*(1+1/x)
f2 <- function(x) k1 * x^a
x <-seq(from=0.05, to=1, length=200)

matplot(
  x, cbind(f1(x),f2(x)), type="l",
  col=2:3, lty=c(1,2), 
  lwd=c(2,2), ylab="f1(x), f2(x)"
)
legend(
  "topright",
  c("f1(x)","f2(x)"), col=2:3,
  lty=c(1,2), lwd=c(2,2)
)

On peut trouver la racine recherchée grâce à la fonction uniroot.

f1 <- function(x) k0*(1+1/x)
f2 <- function(x) k1 * x^a
f  <- function(x) f2(x) - f1(x)

racine <- uniroot(f, lower = 0, upper = 1)

La fonction uniroot() ne retourne jamais qu’une seule racine, dans l’intervalle spécifié.

Lorsque plusieurs racines sont susceptibles d’exister, il est préférable d’utiliser une autre fonction, uniroot.all() qui retourne, elle, toutes les racines existantes dans l’intervalle indiqué. Cette fonction se trouve dans le package rootSolve.

Stabilité d’une équation différentielle à une variable

  1. Implémenter sous R la résolution de l’équation différentielle suivante.

\[ x'(t) = (1 - x) c x^a s - x c (1 - x)^a (1 - s) \]

Avec les contraintes :

  • \(c > 0\)
  • \(a > 1\)
  • \(s \in ]0 ; 1[\)

Et conditions initiales :

  • \(c = 3\)
  • \(a = 2\)
  • \(s = 0.1\)
  • \(x(0) = 0.5\)
# ceci était ma première idée mais elle n'est pas compatible avec
# la fonction `ode`
f_from_constants <- function(c, a, s){
  function(x){
    (1 - x) * c * x^a * s - x * c * (1 - x)^a * (1 - s)
  }
}
myfunc <- f_from_constants(3, 2, 0.1)


h <- 0.01
t_0 <- 0
t_f <- 10

temps <- seq(from = t_0, to = t_f, by = h)
parametres <- c(c=3, a=2, s=0.1)
cond_init <- c(x=0.0001)

derivee.solver <- function(t, condinit, parametres){
  #' Function to be pased to deSolve::ode()
  with(as.list(c(condinit, parametres)),{

}

  #' Generator to yield the derivative fixing the parameters

}

Oui, en analysant l’équation différentielle on peut observer qu’elle est composée de la différence de deux termes contentant \(1 - x\) et \(x\), alternant celui qui est élevé à la puissance \(a\).

Réponse : Nous avons donc que la solution est une constante lorsque \(x = 0 \; \vee x = 1\).

Si ces valeurs existent, que pouvez-vous dire de la limite de la solution de l’équation (1) (lorsque \(t \rightarrow \infty\)) pour des valeurs de conditions initiales proches des valeurs déterminées à la question précédente ?

Elle est constante, si l’on intègre numériquement à partir d’une valeur nulle, le résultat sera nul aussi.

Stabilité d’un système d’équation différentielles à deux variables

Implémenter sous R la résolution du système d’équation différentielle suivant

  • \(x'(t) = -y\)

  • \(y'(t) = x\)

h <- 0.01
t_0 <- 0
t_f <- 10

temps <- seq(from = t_0, to = t_f, by = h)
parametres <- c(c=3, a=2, s=0.1)
cond_init <- c(x=1, y=20)

# TODO : find how to do non-standard evaluation
# providing a symbol or string with the formula
# in order to generalise these functions.
derivee.solver <- function(t, condinit, parametres){
  #' Function to be pased to deSolve::ode()
  with(as.list(c(condinit, parametres)),{
      dx <- -y
      dy <- x
      return(list(c(dx, dy)))
  })
}

xy <- ode(y = cond_init, times = temps, func = derivee.solver, parms = parametres)
xy %>% as.data.frame() %>% tibble() %>% 
  ggplot(aes(x=time)) + 
    geom_line(aes(y=x, colour="x")) +
    geom_line(aes(y=y, colour="y"))

Existe-t-il une ou des valeurs de conditions initiales (valeurs fixées de \(x(0)\), \(y(0)\)) pour lesquelles la solution du système (2) est constante?

Oui, \((x, y)(t_0) = (0, 0)\).
